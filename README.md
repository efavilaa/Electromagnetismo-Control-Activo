# Electromagnetismo-Control-Activo
Simulación en MATLAB de confinamiento magnético activo de un protón dentro de un toroide de 16 cables. Incluye cálculo del campo magnético, fuerza de Lorentz, control sectorizado de rescate y comparación entre los métodos numéricos Euler, RK2 y RK4 para evaluar estabilidad y conservación de energía.
